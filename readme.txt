== Description ==

This plugin performs a number of checks on your WordPress install to detect
common configuration errors and known issues.

There are currently tests in the following categories:
 * PHP Configuration

The plugin will provide feedback in the form of a list of errors which need
to be resolved quickly and then a list of recommendations which are things
to take care of in the longer term.

== Installation ==

1. Upload to your plugins folder, usually `wp-content/plugins/`
2. Activate the plugin on the plugin screen.
3. Check the health of your WordPress install at Tools > Health Check
